In the above example capability capName will be added to the desired capabiltiy for all the tests. Your custom capabilities provider
    should be defined in the SELION_SELENIUM_CUSTOM_CAPABILITIES_PROVIDER configuration parameter as follows.

  
mvn test -DSELION_SELENIUM_CUSTOM_CAPABILITIES_PROVIDER="com.mycompany.test.UserDefinedCapabilities"


  
If more than one custom capabilities providers are required, please separate the fully qualified class names with commas.
