Tapping into WebDriver events using SeLion

How to tap into events generated by WebDriver using SeLion.

Introduction

WebDriver events are primarily those events which are related to some action that Selenium WebDriver performs on the web page or on a native application such as "click" (or) "type" etc., Selenium's answer to tapping into such events is via the EventFiringWebDriver. But there are a couple of issues with using the EventFiringWebDriver.

  1. It does not work with Actions
  2. Selenium dev community are likely to deprecate this class out of the Core libraries and perhaps house it outside of Selenium in a maybe different form.
