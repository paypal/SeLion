PageYAML for iOS automation

This section is focused on SeLion PageYAML to facilitate automation for iOS apps. Concepts outlined in this section build on those described in SeLion PageYAML for web flow.

Contents of a PageYAML File

A PageYAML file that represents a page (or screen) in an iOS application is made up of the following parts.

  • baseClass - The value for this key represents the fully qualified name of a class that will serve as the base class to inherit attributes and/or capabilities from. If your page is new and is not customizing an existing page, then you should specify the value com.paypal.selion.testcomponents.BasicPageImpl which by default is the class within SeLion that provides all the basic operations that are required for any given iOS page.
  • platform - The value for this key represents the platform the YAML is intended for. By default, if nothing is specified it is taken as "web". For platforms like iOS it must be specified as "ios".
  • elements - The value for this key is a set of key/value pairs that represent elements in a page.
    • A unique name for identifying a particular element and type in the page. This name would end up as the name of the member variable of the generated class and maps to the element. The rules for naming locators/selectors are as follows:
      • <prefix>UIAElement for basic UI element.
      • <prefix>UIAButton for Button UI element.
      • <prefix>UIAAlert for Alert UI element.
      • <prefix>UIANavigationBar for Navigation Bar UI element.
      • <prefix>UIAPicker for Picker UI element.
      • <prefix>UIASlider for Slider UI element.
      • <prefix>UIAStaticText for Static Text UI element.
      • <prefix>UIATableView for Table View UI element.
      • <prefix>UIATextField for Text Field UI element.

      Here prefix would be a concise, contextual and meaningful name of your choice.

      • The list of supported iOS elements can be found here.

      For including iOS custom elements refer to the section Configuration Parameters for Customizing the Code Generator Maven Plugin as described in SeLion Code Generator.
