Working with SeLion element events

How to hook into element events provided by SeLion.

Introduction

SeLion element events are primarily those which are related to some action that you perform through elements provided by SeLion such as "click" (or) "select" etc.

Getting Started with Element Event Handling in SeLion

SeLion offers support to hook into events that are generated by SeLion elements. These steps can be followed to start hooking into events using SeLion. As an example we are going to create a DebugScreenshotListener which takes a screenshot and adds it to the report before any element is clicked on.

  1. Create a new listener class by implementing the SeLion exposed interface ElementEventListener. Or extend the AbstractElementEventListener if you don't want to implement all methods.
